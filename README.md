Detector de Sono e Distração no Trânsito

Este projeto utiliza visão computacional para detectar a presença de um rosto em tempo 
real através da webcam, como etapa inicial de um sistema de monitoramento de sono,
cansaço e distração no trânsito.

Tecnologias

● Python

● OpenCV

Como executar
1. Instale as dependências no terminal da sua IDE:
pip install opencv-python
2. Execute o programa;
3. Pressione q para encerrar.
   
Funcionalidades parciais

● Captura de vídeo via webcam

● Detecção de rosto em tempo real
